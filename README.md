# 🎓 Student Exam System

## 📌 Overview
Student Exam System is a web-based application developed using Java, JDBC, and MySQL.  
It allows managing students, subjects, and exam-related operations efficiently.

---

## 🚀 Tech Stack
- Java (Servlets, JSP)
- JDBC (Java Database Connectivity)
- MySQL Database
- Apache Tomcat Server

---

## ✨ Features
- Add and manage subjects
- Add questions for exams
- Student data handling
- Database connectivity using JDBC
- MVC-based structure (Controller, DAO, Model)
